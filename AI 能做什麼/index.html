給老師的 AI 工作夥伴介紹

我是 Claude，接上 Antigravity 之後，可以把想法變成可用的課堂工具。

你可以把我想成會一起備課、寫程式、整理資料、做網頁、檢查錯誤的助教。Antigravity 則像是我的工作室，讓我能讀檔案、改程式、跑測試、開瀏覽器檢查成果，把「我幫你想」推進到「我幫你做出來」。

1. 我是誰？結合 Antigravity 後能做什麼？

Claude

會思考與溝通的 AI 助手

我可以幫你拆解需求、改寫文字、設計活動、產生範例、整理表格，也能用老師聽得懂的方式說明技術問題。

Antigravity

讓 AI 動手工作的開發環境

它提供檔案、終端機、瀏覽器、程式專案與 AI agent 工作流程。我能在裡面建立網頁、修改程式、執行檢查，並把結果開給你看。

一起使用

從一句話到一個作品

你說「做一個英語單字配對遊戲」，我可以規劃畫面、寫 HTML/CSS/JavaScript、測試互動，再調整成適合學生使用的版本。

2. 我能做的事，以及老師日常例子

出考卷與題庫

依照課本範圍產生選擇題、填充題、閱讀題、素養題，並附答案與解析。例如：「幫我出國小五年級分數加減 20 題，難度分三層。」

設計課程活動

把課本內容轉成分組任務、闖關活動、討論題或學習單。例如：「把自然課『水循環』設計成 40 分鐘探究活動。」

教材製作

製作投影片大綱、講義、學習單、閱讀文本、圖片提示語與互動網頁。例如：「做一份三年級成語教學學習單，含例句與小測驗。」

班級經營

協助設計座位規則、獎勵制度、衝突處理流程、班會討論單。例如：「幫我寫一份適合高年級的合作小組公約。」

家長聯絡

把事情說得清楚、禮貌、具體，避免情緒化。例如：「幫我寫一封通知家長孩子作業長期缺交的訊息，語氣溫和但明確。」

差異化教學

把同一份教材改成基礎、普通、挑戰三種版本。例如：「把這篇閱讀文本改成低、中、高三種難度。」

評量規準

建立 rubrics、檢核表與回饋句。例如：「做一份口頭報告評分規準，分成內容、表達、合作三項。」

資料整理

把學生回饋、測驗結果或會議紀錄整理成重點、表格與下一步。例如：「把這些學習單回饋分成常見問題與補救教學建議。」

互動作品開發

製作小遊戲、抽籤器、計分板、單字卡、課堂倒數器。例如：「做一個注音符號翻牌配對遊戲，適合低年級。」

行政與文件

協助寫活動企劃、成果報告、會議摘要、公告文案。例如：「幫我把校外教學流程整理成家長通知單。」

3. 老師最值得裝的 Antigravity 插件 / 擴充功能

第一優先

Live Server

做 HTML、CSS、JavaScript 教材或小遊戲時，可以即時在瀏覽器預覽。老師改一點內容就能看到結果，最適合課堂展示與快速修正。

寫網頁必備

Prettier

自動整理程式碼格式，讓學生與老師都比較看得懂。當班上一起做專案時，格式一致會少很多除錯痛苦。

降低錯字成本

Code Spell Checker

檢查英文變數、檔名與教材文字中的拼字。做英語教材、網頁標題、按鈕文字時特別有用。

管理作品

GitLens 或 Git Graph

如果老師想保存每次修改、比較學生作品版本，這類 Git 視覺化工具很方便。可以看出誰改了什麼，也能回到較早版本。

AI 加值

MCP 相關擴充與可信任服務

若學校有雲端硬碟、資料庫或其他教學系統，可用 MCP 連接外部工具。不過要先確認資安、帳號權限與學生個資規範。

4. 怎麼跟我說話，回答會最好？

Do：給我清楚的教學情境

  • 說明年級、科目、單元、時間長度與學生程度。
  • 範例：「五年級數學，分數加減，40 分鐘，要有練習與出口票。」

Don't：只說「幫我做教材」

  • 需求太大時，我會猜很多，成果容易不貼近你的班級。
  • 可以先給一個小目標，再逐步加功能。

Do：指定格式與限制

  • 告訴我要表格、條列、學習單、簡報大綱或網頁。
  • 也可以說：「不要太難、字數 300 內、可直接印出。」

Don't：把機密或個資直接丟進來

  • 學生姓名、電話、身分證字號、詳細家庭狀況都應先去識別化。
  • 可以改成「學生 A」「家長 B」再請我協助溝通文字。

Do：要求我先問問題或給版本

  • 範例：「先問我 3 個問題再開始。」
  • 範例：「給我保守版、活潑版、挑戰版三種。」

Don't：一次要求完美

  • 比較好的流程是先做草稿，再說「再短一點」「更適合低年級」「改成遊戲」。
  • 我很適合陪你反覆修，而不是一次猜中所有細節。
